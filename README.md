# MNIST

Небольшой проект для распознавания рукописных цифр от 0 до 9.

Пользователь рисует цифру в браузере, после чего изображение отправляется на Flask-сервер. Сервер подготавливает картинку и передаёт её в обученную нейронную сеть, которая возвращает предсказанную цифру.

## Что используется

- Python
- Flask
- PyTorch
- Torchvision
- Pillow
- HTML, CSS, JavaScript
- MNIST

## Зависимости

Для работы проекта нужны Flask для запуска веб-сервера, PyTorch и Torchvision для нейронной сети и датасета MNIST, а также Pillow для обработки изображения, которое пользователь рисует в браузере. Все нужные Python-библиотеки перечислены в файле `requirements.txt`.

## Установка

Нужен Python 3.10 или новее.

Установить зависимости:

```bash
pip install -r requirements.txt
```

## Запуск

Запустить приложение:

```bash
python app.py
```

После запуска открыть в браузере:

```text
http://127.0.0.1:5000
```

Если нужно заново обучить модель:

```bash
python train.py
```
